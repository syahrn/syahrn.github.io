Daftar Di IyemBot

a. Install Telegram. Untuk kami masukkan ke Grup Team Operator. Grup tersebut adalah Grup untuk kerja. Silahkan tanya jawab segala jika mengalami kendala.

b. Di Grup terdapat IyemBot. IyemBot merupakan Robot Telegram yang berguna untuk catatan administrasi manajemen. Silahkan masukkan data diri anda ke iyem dengan cara mengirim pesan ke Grup Team Operator dengan kata kunci yang kami sediakan.

register manis nohp email yem

set norek nomorrekeninganda yem

set namabank namabankanda yem

set namarek namarekeninganda yem

Contoh :

register manis 082136544944 triachmadsyahroni@gmail.com yem
